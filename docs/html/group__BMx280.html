Библиотека для работы с BME/BMP280. Позволяет проводить инициализацию, базовую конфигурацию, чтение измеренных данных.

float * h )

Функция для разового измерения и чтения данных с датчика BME280.

Высота вычисляется по формуле \( 29.254 \cdot (T + 273.15) \cdot \log(\frac{pres_{ref}}{pres})\)

Функции измерения данных не универсальны для датчиков BMP/BME280.

[out]pressИзмеренное давление в Паскалях
[out]humИзмеренная влажность в %

◆ BMx280_config()

HAL_StatusTypeDef BMx280_config ( uint8_t T_OS,
uint8_t P_OS,
uint8_t H_OS,
uint8_t STDB,
uint8_t IIRF )

Конфигурация датчика BMP/BME280.

Конфигурирует OVERSAMPLING для датчика температуры, давления и влажности, частоту работы, коэффициент сглаживания для IIR фильтра.

Заметки
Конфигурация датчика влажности актуальна только для BME280. BMP280 не имеет датчика влажности.
Аргументы
[in]T_OSOVERSAMPLING для термометра. Допустимые значения см. в OVERSAMPLING
[in]P_OSOVERSAMPLING для барометра. Допустимые значения см. в OVERSAMPLING
[in]H_OSOVERSAMPLING для гигрометра. Допустимые значения см. в OVERSAMPLING
[in]STDBЧастоты работы. Допустимые значения см. в STANDBY_MODE
[in]IIRFКоэффициент сглаживания для IIR-фильтра. Допустимые значения см. в IIR_FILTER
Возвращает
HAL_StatusTypeDef Результат получения данных по I2C

HAL_StatusTypeDef BMx280_init ( I2C_TypeDef * hi2c_,

Инициализация BME/BMP280.

Заметки
Автоматически определяет адрес датчика в зависимости от SDO. Данная библиотека поддерживает только один подключенный датчик BMP/BME280
Аргументы
[in]hi2c_Экземляр интерфейса I2C, к которому подключен датчик
[in]refPressure_Начальное давление, относительно которого вычисляется высота по давлению
Возвращает
HAL_StatusTypeDef Результат получения данных по I2C

◆ BMx280_normal_measure()

HAL_StatusTypeDef BMx280_normal_measure ( )

Переход в режим измерений.

Производит измерения с частотой STANDBY_MODE, заданной с помощью BMx280_config.
Измерения записывает во внутреннюю память, которые можно считать функцией BMP280_get_measure или BME280_get_measure.
