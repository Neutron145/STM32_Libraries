\chapter{Главная страница }
\hypertarget{index}{}\label{index}\index{Главная страница@{Главная страница}}
~\newline
 Документация библиотек датчиков ракеты "{}Пересмешник"{} на STM32 с использованием LL и HAL. На данный момент проект находится в активной разработке, пишется документация и производится рефакторинг кода и структуры библиотеки. Библиотека предполагает использование по одному экземпляру датчиков в ракете.\hypertarget{index_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{Как подключить библиотеку к своему проекту?}{Как подключить библиотеку к своему проекту?}}\label{index_autotoc_md0}
Скачать файлы .c и .h для необходимого модуля. Переместить .c файл в директорию Core/\+Src, .h файл в директорию Core/\+Inc. В main.\+c прописать \#include "{}$<$название модуля$>$.\+h"{}. После чего библиотеку можно использовать в Вашем проекте.\hypertarget{index_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Как читать комментарии исходного кода?}{Как читать комментарии исходного кода?}}\label{index_autotoc_md1}
В ходе разработки библиотеки в заголовочных файлах (.h формат) был написан самодокументируемый код, который можно читать как в настоящей документации, так и в самих файлах. Все коментарии писались по стандарту Javadoc, все его ключевые слова начинаются с символа \textquotesingle{}@\textquotesingle{}. В исходных файлах (.c формат) написаны дополнительные пояснения к реализации функций. ~\newline
 В самом начале файла пишется блок документации самого файла, далее идет блок документации модуля библиотеки. ~\newline
 После этого объявляются макросы, разбитые по группам ключевым словом defgroup. ~\newline
 За макросами объявляются функции, перед которыми пишутся блоки документации этих функций. ~\newline


Ключевые слова Javadoc, используемые в документации\+: ~\newline

\begin{DoxyItemize}
\item brief \{описание\} -\/ краткое описание элемента ~\newline

\item details \{описание\} -\/ детальное описание элемента ~\newline

\item defgroup $<$\+ID$>$ \{Заголовок\} -\/ объявление группы ~\newline

\item param $<$имя параметра$>$ \{описание параметра\} -\/ описание параметра функции ~\newline

\item retval $<$имя$>$ \{описание\} -\/ описание возвращаемого значения ~\newline

\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{Как пользоваться документацией к библиотеке?}{Как пользоваться документацией к библиотеке?}}\label{index_autotoc_md2}
Документация локальная, хранится в виде HTML страниц в папке docs. Открывается через файл index.\+html. ~\newline
 Разбита по модулям библиотеки, находящиеся в заголовке \textquotesingle{}Topics\textquotesingle{} и по файлам, находящимся в заголовке \textquotesingle{}Файлы\textquotesingle{}. ~\newline
