Конфигурация высокочастотных фильтров для гироскопа и акселерометра

Повзоляет настроить высокочастотный фильтр (HPF) для гироскопа и акселерометра.
Высокочастотный фильтр пропускает резкие движения, отфильтровывает медленные движения и дрейф.

Аргументы
[in]g_HPFРежим работы фильтра HPF для гироскопа. Принимает значения макросов режима работы фильтра.
[in]g_HPF_frequencyЧастота фильтра HPF для гироскопа. Принимает значения макросов частоты фильтра для гироскопа.
[in]a_HPFЧастота и режим работы HPF для акселерометра. Принимает значения макросов фильтра high-pass frequency для акселерометра.
Возвращает
HAL_StatusTypeDef Результат получения данных по I2C

uint8_t signs )

Конфигурация ориентации датчика

Конфигурируется порядок расположения осей системы координат и их знаки относительно заводской системы координат.

Заметки
Стандартная система координат датчика совпадает с осями, размеченными на корпусе датчика и имеет порядок X, Y, Z с положительными направлениями.
Аргументы
[in]orientПорядок расположение осей системы координат. Принимает значения макросов порядка расположения осей системы координат.
[in]signsЗнаки осей в системе координат. Принимает значения макросов направлений осей системы координат.
Возвращает
HAL_StatusTypeDef Результат получения данных по I2C

Конфигурация режима работы датчика

Конфигурирует частоту работы акселерометра и гироскопа. От них зависит качество работы фильтров и измеренных данных, а также энергопотребление датчика.

Аргументы
[in]a_ODRЧастота работы акселерометра. Принимает значения макросов частоты и питания датчика.
[in]g_ODRЧастота работы гироскопа. Принимает значения макросов частоты и питания датчика.
