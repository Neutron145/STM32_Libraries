\doxysubsubsection{Подробное описание}
Конфигурация IIR-\/фильтра для барометра 

Чем выше коэффициент -\/ тем меньше шумов, но медленнее реакция на изменения. ~\newline
 Зависимость коэффициента фильтра на скорость реакции отражена на графике\+:  То есть без фильтра при следующем измерении будет получено 100\% от резкого изменения сигнала. При коэффициенте 16 после резкого изменения сигнала будет получено 90\% сигнала после 32 измерений.
