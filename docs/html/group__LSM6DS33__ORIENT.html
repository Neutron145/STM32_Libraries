Подробное описание

Порядок расположения осей системы координат в массиве полученных измерений

Значения передаются в функцию LSM6DS33_config_orientation для настройки порядка осей. Стандартный порядок - X, Y, Z. В соответствии с осями, размеченными на датчике.

  • Создано системой 1.15.0
